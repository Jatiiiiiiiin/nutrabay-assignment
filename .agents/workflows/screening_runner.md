To execute a complete AI Resume Screening benchmark audit on the Standalone Nutrabay interface wrapper:

1. **Start Framework Services**
   - Ensure the React dev server is mapped: `npm run dev` in workspace root.
   - Open standard local viewer (typically `http://localhost:5173`).

2. **Supply Target Description Boundaries**
   - Paste corresponding Job Description requirements inside Bucket 1 in layout grid frame.

3. **Supply Target Candidate Boundaries**
   - Drag or select target `.pdf` candidates directly to Bucket 2 in dashboard viewer.

4. **Trigger Scoring Algorithm triggers**
   - Press **Run AI Screening** button bucket action.
   - View structured analytic arrays populated natively inside audit frame layouts beneath viewport.
